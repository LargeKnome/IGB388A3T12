Universal Render Pipeline upgrade
---------------------------------

Supported version: 7.1.8 and up only!

1) Import Urban Underground pack to your URP project, and run UU_SRP_Upgrade pack!
2) Enjoy!

High Definition Render Pipeline upgrade
---------------------------------------

Supported version: 7.1.8 and up only!

1) Import Urban Underground to your configured HDRP project!
2) Run the auto-upgrade tool on it (Edit/Render pipeline/project materials to HDRP)
3) When done, run UU_HDRP_Upgrade unitypackage!
4) Demo scene lighting, volumes and probes will need to be set up and adjusted, please refer to Unity's HDRP guidelines!
